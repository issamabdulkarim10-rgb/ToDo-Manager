# ToDo Manager (WPF, MVVM)

Eine einfache ToDo-Anwendung in WPF, umgesetzt mit dem MVVM-Pattern.

## Funktionen

- Aufgaben erstellen und verwalten
- Aufgaben als erledigt markieren
- Trennung in offene und erledigte Aufgaben
- Priorität und Fälligkeitsdatum festlegen

## Architektur

Das Projekt basiert auf dem MVVM-Pattern:

- Model → Datenstruktur
- ViewModel → Anwendungslogik
- View → Benutzeroberfläche (WPF)

## Technologien

- C#
- WPF (.NET)
- MVVM
- ObservableCollection / ICollectionView
